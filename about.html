About Ovechkin Cam

This app is a catalog of Ovechkin goals from the first to the last (898).

Our sequential index allows users to search through Ovechkin goal highlights in a novel and useful way.

This web address accepts queries that link directly to specific goals (?1 to ?898) or a set of search results (?sean-burke).

Search Tips:

  • "Empty Net" for Empty Net Goals
  • "First" for Season Starting Goals
  • "GWG" for Game Winning Goals
  • "Last" For Season Ending Goals
  • "Overtime" for Overtime Goals
  • "PPG" for Power Play Goals
  • "SHG" for Shorthanded Goals
