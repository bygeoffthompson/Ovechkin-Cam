About Ovechkin Cam

Ovechkin Cam is a catalog of goal videos from first to last (911) that also includes All Star Game (8), Playoff (77), IIHF (35), KHL (19), and Olympic (8) goals for a total of 1,058.

Our searchable sequential index allows users to find Ovi goals in a new and useful way.

Ovechkin.cam accepts a URL query to retrieve specific goals (?1 to ?911) or a set of filtered search results (?fleury+golden-knights). A type filter can be added to the query to further reduce results (?fleury+golden-knights|playoff).
